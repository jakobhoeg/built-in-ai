The `@built-in-ai/transformers-js` package makes it incredibly easy to run inference with [Hugging Face Transformers](https://github.com/huggingface/transformers.js) models in the browser with the Vercel AI SDK v6. No complex Web Worker setup, no manual streaming.

### What's Included

- **Language Models**: Text generation with SmolLM, Qwen, Llama, and more
- **Text Embeddings**: GTE, MiniLM, and other embedding models
- **Vision Models**: SmolVLM and other multimodal models
- **Transcription**: Whisper models for speech-to-text
- **Tool Calling**: Function calling support
- **Agents**: Multi-step autonomous workflows
