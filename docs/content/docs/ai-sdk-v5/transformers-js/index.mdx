Run popular open-source [Hugging Face Transformers](https://huggingface.co/models) models directly in the browser or server-side using [Transformers.js](https://github.com/xenova/transformers.js).

<Callout type="info">
  This library is in active development. Expect frequent updates.
</Callout>

## Overview

The `@built-in-ai/transformers-js` package makes it incredibly easy to run Hugging Face models in the browser with the Vercel AI SDK. No complex Web Worker setup, no manual streamingâ€”just a few lines of code.

### Key Features

- **Simple API**: What takes 200+ lines with raw Transformers.js takes just 15 with this package
- **Full AI SDK integration**: Works seamlessly with `useChat`, `streamText`, and more
- **Web Worker support**: Built-in worker handler for off-main-thread execution
- **Progress tracking**: Automatic download progress for better UX
- **Server-side support**: Works in Node.js too, not just browsers

### What's Included

- **Language Models**: Text generation with SmolLM, Qwen, Llama, and more
- **Text Embeddings**: GTE, MiniLM, and other embedding models
- **Vision Models**: SmolVLM and other multimodal models
- **Transcription**: Whisper models for speech-to-text
- **Tool Calling**: Function calling support
